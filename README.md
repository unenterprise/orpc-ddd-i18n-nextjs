# orpc-ddd-i18n-nextjs
This repository contains an example of how to integrate **orpc-ddd-i18n** with a **Next.js** application. It demonstrates the use of Domain-Driven Design (DDD) principles along with internationalization (i18n) support in a modern web application.

## Features
- Domain-Driven Design (DDD) architecture
- Internationalization (i18n) support
- Integration with Next.js framework
- Example domain models and services

## Getting Started
Do not clone the repository. Instead, use the following command to create a new project based on this example:

   ```bash
   npx degit unenterprise/orpc-ddd-i18n-nextjs my-project
   ```
